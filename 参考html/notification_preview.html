showNotification (Standard)

This method displays a standard Android notification using the parameters you provide.

  • title, text: The main title and content text of the notification.
  • ongoing: If true, the notification cannot be dismissed by the user.
  • autoCancel: If true, the notification is cancelled when the user taps it.

showCustomNotification (New)

This new method allows you to build a notification with a custom layout by providing a list of view objects.

  • title, text: These still provide the base title and content for the notification, visible in the standard header.
  • views: A List<LiveUpdateView> that defines the custom layout. The plugin currently supports:
    • LiveUpdateTextView(text: '...'): Adds a simple line of text.
    • LiveUpdateScrollingTextView(text: '...'): Adds a line of text that will automatically scroll horizontally (marquee) if it exceeds the notification width.
    • LiveUpdateImageView(imageProvider: ...): Adds an image from a Flutter ImageProvider.

Note: The final appearance of notifications can vary across different Android versions and device manufacturers. This preview is a close approximation.
